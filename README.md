# Frontend: An Introductory Course

Repositorio educativo sobre desarrollo web frontend, desde los conceptos más básicos hasta técnicas avanzadas.

Este proyecto documenta el aprendizaje progresivo de las tecnologías fundamentales del desarrollo web: HTML, CSS, y los conceptos necesarios para construir sitios web modernos y profesionales.

---

## Estructura del Repositorio

Cada módulo contiene:
- Archivos HTML/CSS de ejemplo
- Un `readme.md` con la teoría completa
- Código comentado y explicado

### Módulos Disponibles

#### 1. [Primer HTML](./primer_html/)
**Fundamentos de HTML**
- Estructura básica de un documento HTML
- Etiquetas semánticas principales
- Elementos comunes: títulos, párrafos, listas, tablas, enlaces, imágenes
- Anatomía de una página web

#### 2. [Primer CSS](./primer_css/)
**Introducción a CSS**
- ¿Qué es CSS y para qué sirve?
- Formas de vincular CSS a HTML
- Sintaxis y selectores básicos
- Propiedades fundamentales: colores, tipografía, espaciado
- El Box Model
- Cascada y especificidad

#### 3. [Layout: Flexbox y Grid](./layout_flexbox_grid/)
**Sistemas de Layout Modernos**
- Flexbox: alineación y distribución en una dimensión
- CSS Grid: layouts bidimensionales complejos
- Comparación entre Flexbox y Grid
- Patrones de layout comunes: páginas, dashboards, galerías
- Ejemplos prácticos con código comentado

---

## Objetivo

Este repositorio sirve como:
- **Referencia rápida** de conceptos fundamentales
- **Documentación** de aprendizaje progresivo
- **Base teórica** para desarrollo web frontend

No es un curso con ejercicios, sino una **colección de notas técnicas** que explican los conceptos y tecnologías del desarrollo web.

---

## Próximos Módulos

Los siguientes temas se irán añadiendo progresivamente:

- **Diseño Responsive**: Media queries, mobile-first approach
- **Componentes Web**: Botones, cards, navegación, formularios
- **CSS Avanzado**: Transiciones, animaciones, gradientes, variables CSS
- **Frameworks CSS**: Tailwind CSS, introducción a frameworks modernos
- **Generadores de Sitios Estáticos**: Hugo, Markdown, templates
- **Buenas Prácticas**: Organización de código, accesibilidad, performance

---

## Cómo Usar Este Repositorio

1. **Navega** por los módulos en orden progresivo
2. **Lee** el `readme.md` de cada módulo para entender la teoría
3. **Examina** los archivos de ejemplo para ver la aplicación práctica
4. **Experimenta** modificando el código en tu entorno local
